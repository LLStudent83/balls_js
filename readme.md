# Игра "Поймай все шарики"

## Описание

Это интерактивная браузерная игра, в которой нужно за максимально короткое время поймать все шарики, избегая столкновений с "злым кругом".

## Цель проекта

Научиться использовать современные инструменты и технологии веб-разработки, такие как:

- **Shadcn** - библиотека готовых UI компонентов
- **Tailwind CSS** - утилитарный CSS фреймворк  
- **Zustand** - легковесное решение для управления состоянием
- **Zod** - схема валидации TypeScript
- **Orval** - генератор клиентского кода для OpenAPI
- **React Router v7** - маршрутизация для React
- **Vite**: - сборщик
